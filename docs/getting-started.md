# Getting Started

## Requirements

- Intel Quartus Prime Lite
- USB-Blaster programmer
- USB-C cable for power

## Powering the Board

Both boards are powered through the onboard USB-C connector.

The USB connector is used only for power delivery.

## Programming

1. Connect an external USB-Blaster to the onboard 2x5 JTAG header.
2. Open Quartus Prime Lite.
3. Compile the design.
4. Open Programmer.
5. Detect the FPGA.
6. Upload the generated bitstream.
